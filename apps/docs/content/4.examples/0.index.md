# Examples

Practical, ready-to-use code snippets for the most common Owlat integrations. Examples use the TypeScript SDK, with curl variants where a direct HTTP call applies.

## Sending Emails

- [Welcome Email](/examples/welcome-email) — send a personalized welcome email when a user signs up
- [Billing Email](/examples/billing-email) — send a receipt with an invoice PDF attached

## Events & Automation

- [Event Automation](/examples/event-automation) — trigger automations with custom events
- [Contact Sync](/examples/contact-sync) — sync contacts from your database to Owlat

## Infrastructure

- [Webhook Handler](/examples/webhook-handler) — handle delivery webhooks with signature verification
- [Multilingual Email](/examples/multilingual-email) — send emails in the recipient's preferred language
