# Code of Conduct

This project follows the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
the de-facto standard code of conduct for open-source communities. By
participating — in issues, pull requests, discussions, or any other project
space — you agree to uphold it.

The short version: be respectful, be constructive, and treat everyone with
empathy. Harassment, discrimination, personal attacks, and other behavior
that would be unacceptable in a professional setting are unacceptable here.

## Reporting

Report violations to **security@owlat.app**. Reports are reviewed promptly
and handled with discretion; the reporter's identity is kept confidential.
This is the same address used for security reports (see
[SECURITY.md](SECURITY.md)) — both kinds of report reach a human maintainer
directly.

## Enforcement

Maintainers are responsible for clarifying and enforcing these standards and
will take appropriate, fair corrective action — from a private warning up to
a permanent ban from project spaces — in response to behavior they deem
inappropriate, threatening, offensive, or harmful, following the
[Contributor Covenant enforcement guidelines](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines).
